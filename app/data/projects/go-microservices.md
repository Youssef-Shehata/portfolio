# Go Microservices

A collection of microservices built with Go and gRPC.

## Features

- gRPC communication
- Docker containerization
- Kubernetes deployment
- Prometheus metrics
- Jaeger tracing

## Architecture

The system consists of:
- API Gateway
- Auth Service
- User Service
- Product Service
- Order Service
